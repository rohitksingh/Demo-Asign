# Notely App

## Features                
Currently the following functions are implemented:

 - Basic add, edit, delete functionalities
 -  Material design interface               
 - Navigation Drawer             
 - Swipe to delete            
 - RecyclerView          
 - Custom menu 
 - Object  oriented back-end 
 - Serialization and Deserialization
